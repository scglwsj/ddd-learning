## Description

learn DDD by parking in TypeScript

## Environments

[Node.js](https://nodejs.org/en/) `v13.9.0`

[TypeScript](https://www.typescriptlang.org/) `Version 3.8.2`

[npm](https://www.npmjs.com/) `6.14.1`

[yarn](https://yarnpkg.com/) `1.22.0`

## Installation

```bash
yarn
```

or

```bash
npm i
```

## Testing

```bash
yarn test
```

or

```bash
npm test
```

## Formatting

```bash
yarn run format
```

or

```bash
npm run format
```
